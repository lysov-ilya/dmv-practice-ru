# Тест DMV Калифорнии на русском

Бесплатный русскоязычный тренажёр для подготовки к письменному тесту на водительские права California DMV категории C.

**[Открыть тренажёр →](https://dmv-practice-ru.vercel.app/)**

## Что внутри

- 438 вопросов для самостоятельной практики.
- Официальные русскоязычные пробные тесты California DMV.
- Дополнительные вопросы по темам справочника водителя.
- Случайный порядок вопросов и повторение ошибок, сохранённых в браузере.

## Важно

Это независимый учебный проект, а не официальный сайт California DMV. Вопросы на реальном экзамене могут отличаться. Перед тестом изучите актуальный [русский справочник водителя California DMV](https://www.dmv.ca.gov/portal/file/california-driver-handbook-russian-pdf/) и [официальные образцы тестов](https://www.dmv.ca.gov/portal/driver-education-and-safety/educational-materials/sample-driver-license-dl-knowledge-tests/).

## Запуск локально

Проект не требует сборки или зависимостей:

```bash
python3 -m http.server 8000
```

После запуска откройте <http://localhost:8000>.

## Публикация

Сайт размещён на Vercel. Каждый push в ветку `main` автоматически создаёт production-деплой.
